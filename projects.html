Dynamic Structure Autoencoder

● Public
MLPyTorchOptimization

A self adjusting neural network architecture that prunes and shapes during training. The model relies on a path-integral formulation of entropy included in the loss function to promote a bottleneck structure. Tested on CIFAR10 and MNIST datasets. Implemented in PyTorch and Numpy.

Graph-Coloring via Markov Chains

● Private
Algorithms MCMC Python

Algorithm + Visualization of a Markov Chain Monte Carlo method for sampling and estimating the number of colorings a graph can have in polynomial time. Video and code done in Python with libraries: Manim, Numpy, Matplotlib. Analysis of the speed of convergence is also present.
